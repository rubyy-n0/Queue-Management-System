# Queue Management System

A modern **Queue Management System** that helps businesses organize customer queues efficiently, reduce waiting time, and improve service quality. The system provides real-time queue updates using Firebase and is deployed on Google Cloud for reliable performance.

---

## Features

- Real-time queue updates
- Customer queue registration
- Queue number generation
- Live queue status display
- Responsive and user-friendly interface
- Cloud-based backend using Firebase
- Fast and secure data synchronization

## Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/Queue-Management-System.git
```

2. Open the project folder.

```bash
cd Queue-Management-System
```

3. Configure Firebase.

- Create a Firebase project.
- Enable Firestore Database.
- Enable Authentication (if required).
- Replace the Firebase configuration inside `firebase-config.js`.

4. Run the project locally.

Simply open `index.html` using Live Server in Visual Studio Code.
